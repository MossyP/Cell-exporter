# Cell Exporter for AE

Blender 4.2用アドオンです。コレクションを「セル」ごとに連番画像として書き出し、After Effects用の自動インポートスクリプト（`.jsx`）も同時生成します。各セルは自動でレイヤーとして読み込まれ、順序も正しく重ねられます。

## 特徴

- 任意のコレクションを「Cell」として登録（`Cell_A`, `Cell_B`, `Cell_C`...）
- 各セルをPNGまたはEXR連番として個別フォルダに書き出し
- 各セルのレンダリング中もライト・カメラは常に表示されるよう自動制御
- 背景を透明（アルファ）で出力
- After Effects用インポートスクリプト（`.jsx`）を自動生成：
  - レンダー設定に合わせたコンポジションを自動作成
  - 各セルフォルダを連番として自動インポート
  - セルの重ね順はBlender側での設定と同様のものに。
 
## 環境

- Blender 4.2 LTS 以降

## インストール方法

1. `cell_exporter.zip` をダウンロード
2. Blenderで `編集 > プリファレンス > アドオン > インストール...`
3. zipファイルを選択してアドオンを有効化

## 使い方

1. 3Dビューポートのサイドバー（`N`キー）から **Cell Exporter** タブを開く
2. **出力先パス（Output Base Path）** と書き出し形式（PNG / EXR）を設定
3. **＋ボタン** でセルを追加し、それぞれにコレクションを割り当てる
4. ↑↓ボタンで順序を入れ替え（リストの上＝AEでの最上位レイヤー）
5. **Exportボタン** を押すと、各セルがレンダリングされ `import_cells.jsx` が生成される
6. After Effectsで `ファイル > スクリプト > スクリプトファイルの実行...` から `import_cells.jsx` を選択
